The Jinja templating language

With the Jinja templating engine, we can display Python object in our HTML templates.

Our superheroes are in this list:

{{ heroes }}

{{ heroes[0].name }} VS {{ heroes[1].name }}

Jinja supports `for` loops:

{% for hero in heroes %} {{ hero }} {% endfor %}

To make this more readable, we can wrap the hero names in a HTML list:

    {% for hero in heroes %}
  • {{ hero }} has {{ hero.life }} health points left
    • {% endfor %}
